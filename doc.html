Начало работы

В этом блоке собрана подробная документация по работе с Assistant, например установка и конфигурация.

Установка

              
                $ assistant  ——config
                  {
                    “runInBackground”: true, 
                    “highlighting“: true,
                    “prettyTable”: false,
                    “font”: [“Helvetica”, “sans-serif”],
                    “folder”: “~/Desktop“
                  }

Конфигурация

Тут указаны конфигурируемые настройки Assistant для максимаально удобного пользования.

Опция Значение По умолчанию
runInBackground Работа в фоновом режиме после закрытия окна. false
highlighting Выделение синтаксиса. true
prettyTable Выводить данные в форме таблицы. true

Горячие клавиши

    По умолчанию

  • Выйти Ctrl+C
  • Сохранить Ctrl+S
  • Сохранить и выйти Ctrl+D
  • Отмена Ctrl+Z

    Разметка

  • Ctrl+A Вставить ссылку с разметкой
  • Ctrl+I Вставить изображение с разметкой
  • Ctrl+V Вставить видео с YouTube
  • Ctrl+T Вставить таблицу
